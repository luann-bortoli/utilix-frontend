# Utilix - Hub de ferramentas digitais

O Utilix é um hub de ferramentas rápidas e convenientes voltadas para o uso diário.
O objetivo do projeto é centralizar utilitários como calculadoras, conversores e geradores.

# Features
  - Calculadora de juros simples.
  - Calculadora de juros composto.
  - Gerador de senhas seguras.
  - Contador de caracteres.

# Tecnologias Utilizadas
- React.js
- Vite
- JavaScript
---
## Como Rodar o Projeto
  # Clonar o repositório
  git clone https://github.com/luann-bortoli/utilix-digital-tools

  # Entrar na pasta
  cd utilix

  # Instalar dependências
  npm install

  # Rodar localmente
  npm run dev
